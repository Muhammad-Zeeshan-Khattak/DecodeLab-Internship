# DecodeLabs AI Internship

This repository contains projects completed during my Artificial Intelligence Internship at DecodeLabs.

The purpose of this internship is to learn and apply fundamental Artificial Intelligence concepts through hands-on projects. Each project focuses on problem-solving, logical thinking, programming, and AI development techniques.

## Project 1: Rule-Based AI Chatbot

A Java-based chatbot that uses if-else decision-making and control flow to respond to predefined user inputs.

### Features

- Greeting responses
- Exit commands
- Date and time information
- AI and programming knowledge
- Mathematical operations
- Prime number and palindrome checking
- Random jokes and fun facts
- Motivation and study tips
- Rule-based conversation handling

### Technologies Used

- Java
- Object-Oriented Programming
- Control Flow
- If-Else Logic
- Loops
- Scanner Class

### Internship

This project was developed as part of the Artificial Intelligence Internship Program at DecodeLabs.
